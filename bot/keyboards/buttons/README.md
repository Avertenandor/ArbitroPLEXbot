# Buttons Module Structure

Этот модуль содержит все текстовые константы для кнопок бота, организованные в логические подмодули.

## Структура модуля

### Пользовательские кнопки (User-facing)

- **`main_menu.py`** - Кнопки главного меню пользователя
  - `MainMenuButtons`: Баланс, Депозиты, Выводы, Рефералы, Поддержка, Настройки и т.д.

- **`deposits.py`** - Кнопки депозитов
  - `DepositButtons`: Кнопки уровней депозитов, смена кошелька

- **`withdrawals.py`** - Кнопки выводов средств
  - `WithdrawalButtons`: Вывод средств, история
  - `WithdrawalSettingsButtons`: Настройки выводов

- **`referrals.py`** - Кнопки реферальной системы
  - `ReferralButtons`: Мои рефералы, статистика, заработок

- **`settings.py`** - Кнопки настроек
  - `SettingsButtons`: Профиль, кошелек, уведомления
  - `ContactButtons`: Обновление контактов
  - `NotificationButtons`: Переключатели уведомлений

- **`transactions.py`** - Кнопки истории транзакций
  - `TransactionButtons`: Типы транзакций, фильтры, экспорт

- **`auth.py`** - Кнопки авторизации
  - `AuthButtons`: Оплата, начало работы, проверки
  - `AppealButtons`: Апелляции для заблокированных пользователей

### Административные кнопки (Admin)

- **`admin.py`** - Основные админ-кнопки
  - `AdminButtons`: Главная админ-панель
  - `AdminUserButtons`: Управление пользователями
  - `AdminWalletButtons`: Управление кошельками
  - `AdminManagementButtons`: Управление админами

- **`admin_deposits.py`** - Управление депозитами
  - `AdminDepositButtons`: Статистика, уровни, настройки ROI

- **`broadcast.py`** - Рассылка сообщений
  - `BroadcastButtons`: Добавление кнопок к рассылке

- **`blacklist.py`** - Черный список
  - `BlacklistButtons`: Управление черным списком

- **`master_key.py`** - Мастер-ключи
  - `MasterKeyButtons`: Показать/генерировать ключи

- **`user_messages.py`** - Просмотр сообщений
  - `UserMessagesButtons`: Просмотр и управление сообщениями пользователей

- **`roi.py`** - ROI коридоры
  - `ROICorridorButtons`: Настройка коридоров доходности

- **`financial.py`** - Финансовые отчеты
  - `FinancialReportButtons`: История, депозиты, выводы, кошельки

### Общие кнопки (Common)

- **`navigation.py`** - Навигация
  - `NavigationButtons`: Назад, Отмена, Главное меню
  - `PaginationButtons`: Предыдущая/Следующая страница

- **`actions.py`** - Действия и подтверждения
  - `ActionButtons`: Да/Нет, Подтвердить/Отклонить, Отправить

- **`support.py`** - Поддержка
  - `SupportButtons`: Создание тикетов, FAQ, управление обращениями

- **`inline.py`** - Инлайн-кнопки
  - `InlineButtons`: Блокчейн настройки, восстановление финпароля

## Использование

### Обратная совместимость

Все существующие импорты продолжают работать:

```python
# Старый способ (все еще работает)
from bot.keyboards.buttons import MainMenuButtons, AdminButtons

# Новый способ (более явный)
from bot.keyboards.buttons.main_menu import MainMenuButtons
from bot.keyboards.buttons.admin import AdminButtons
```

### Рекомендации

- Для **нового кода** рекомендуется импортировать из конкретных подмодулей
- Для **существующего кода** ничего менять не нужно - все импорты совместимы
- При добавлении новых кнопок добавляйте их в соответствующий подмодуль

## Преимущества новой структуры

1. **Улучшенная читаемость** - легче найти нужные кнопки
2. **Логическая группировка** - связанные кнопки находятся вместе
3. **Меньше merge-конфликтов** - разные разработчики работают с разными файлами
4. **Проще навигация** - файлы по 20-100 строк вместо одного файла на 564 строки
5. **Полная обратная совместимость** - существующий код продолжает работать
